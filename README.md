  - Googleがオープンソースで公開した機械学習システム
  - データの流れを表す図(Data Flow Graph)を使って、計算方法を組み立てられる
  - ある入力を、データフローグラフに従って評価
  - 評価した結果が目的の値(小さい値とか、大きい値とか)になるように、計算式内のパラメータを調整する
  - 上記を繰り返すことで、入力に対して、良さそうな結果を出力する
